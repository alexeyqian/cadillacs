# Reference Folder Template

This folder is for organizing visual and gameplay reference for a Cadillacs & Dinosaurs inspired Pygame beat-em-up project.

Important: do not commit or redistribute copyrighted original game sprites, ROM assets, extracted sprite sheets, music, sound effects, or screenshots unless you have the legal right to use them. Use this folder for private reference, measurements, notes, and your own newly-created assets.

Recommended workflow:

1. Put original-game screenshots or notes in `arcade_screenshots/` only if you have the right to store them privately.
2. Record measurements and gameplay observations in `measurements/` and `animation_notes/`.
3. Put your own newly-created sprites in `own_created_assets/`.
4. Keep final game-ready assets separate from copyrighted reference material.

Suggested filename style:

```text
character_action_direction_frame.png
jack_walk_right_0001.png
enemy_punk_attack_left_0003.png
stage01_city_reference_001.png
```
